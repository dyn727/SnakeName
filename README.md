# snake-噶么

简单的浏览器端贪吃蛇小游戏。

目录结构：

```
snake-噶么/
├── assets/        # 图片、音效等资源文件夹
├── index.html     # 页面骨架（只保留DOM结构）
├── styles.css     # 全部样式代码
├── script.js      # 全部游戏逻辑JS
└── README.md      # 项目说明文档
```

运行方法：

- 直接在浏览器中打开 `index.html`，或使用静态服务器（例如 `http-server`）来预览。

说明：

- 使用方向键控制蛇移动，按空格键开始/暂停。
- 高分保存在 `localStorage` 中。
