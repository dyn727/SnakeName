# snake-game

欢迎来到 `snake-game`，一款轻量级的浏览器贪吃蛇游戏。

## 游戏规则

1. 游戏开始后，蛇会持续移动。
2. 使用方向键 `↑`、`↓`、`←`、`→` 控制蛇头方向。
3. 每吃到一个食物，蛇身会变长一格，同时得分增加 10 分。
4. 吃到食物后，新的食物会随机出现在画布上，并且当前蛇身不能占据该位置。
5. 随着得分增加，游戏速度会逐渐加快，让操作更具挑战性。
6. 当蛇头碰到画布边界，或者撞到自己的身体，游戏结束。

## 操作方式

- `Space` 空格键：开始游戏、暂停/继续。
- `开始` 按钮：开始游戏。
- `暂停` 按钮：暂停或继续当前游戏。
- `重新开始` 按钮：重置游戏，蛇回到起始位置，分数和速度恢复初始值。

## 游戏界面说明

页面包含以下部分：

- 游戏画布：使用 `canvas` 渲染蛇、食物和背景。
- 当前得分：显示本局吃到食物后的累计分数。
- 最高分：从浏览器 `localStorage` 中读取并展示历史最高分。
- 速度显示：当前游戏刷新间隔，数值越小表示蛇移动越快。
- 控制按钮：开始、暂停、重新开始，方便鼠标操作。

## 游戏特点

- 纯前端实现：只需打开 `index.html`，无需后端。
- 本地最高分记录：每局得分高于历史最高分时会自动保存。
- 经典规则：遵循传统贪吃蛇游戏玩法，适合快速体验。
- 速度动态调整：吃到食物后，刷新间隔会降低，游戏节奏更快。

## 项目结构

- `index.html`：页面结构与按钮控制。
- `styles.css`：整体游戏界面样式。
- `script.js`：游戏逻辑、碰撞检测、绘图和键盘事件处理。
- `README.md`：项目说明与玩法介绍。

## 运行方式

- 直接在浏览器中打开 `index.html`。
- 或者使用静态服务器（例如 `http-server`）预览页面。
