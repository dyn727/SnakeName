# snake-game

欢迎来到 `snake-game`，一款基于浏览器的经典贪吃蛇小游戏。本项目使用 `HTML5 Canvas` 渲染游戏画面，配合纯 JavaScript 处理控制逻辑与碰撞检测。

## 游戏目标

你的目标是让蛇尽可能长，吃掉更多食物并获得更高分数。每吃到一个食物，蛇身会变长一节，同时游戏速度会逐渐加快，从而提高挑战性。

## 详细规则

1. 游戏一开始，蛇会按照初始方向自动移动。
2. 玩家可以使用键盘控制方向，但不能立即掉头。
3. 如果蛇头与画布边界碰撞，游戏立即结束。
4. 如果蛇头撞到自己的身体，游戏同样结束。
5. 吃到食物时：
   - 蛇身增长一节；
   - 得分增加 10 分；
   - 随机重新生成一个新食物；
   - 游戏移动速度会稍微加快。
6. 每局结束后，如果本局得分高于当前最高分，则会自动将最高分保存到浏览器 `localStorage`。

## 操作方式

- `↑`、`↓`、`←`、`→`：控制蛇头移动方向。
- `Space` 空格键：开始游戏、暂停或继续当前局。
- `开始` 按钮：开始或继续游戏。
- `暂停` 按钮：暂停当前游戏。
- `重新开始` 按钮：重置游戏并重新开始新一局。

## 界面设置说明

游戏页面由以下区域组成：

- `canvas` 画布：负责显示游戏场景、蛇和食物。
- `得分` 面板：展示当前局得分。
- `最高分` 面板：展示历史最高得分。
- `速度` 面板：显示当前刷新速度，速度值越小表示蛇移动越快。
- `按钮区`：提供开始、暂停、重新开始等操作。

## 玩法特色

- 经典贪吃蛇体验：遵循传统规则，操作简单，适合快速上手。
- 自动加速机制：吃到食物后会提升移动速度，增加游戏紧张感。
- 本地高分记录：使用 `localStorage` 保存最高分，刷新页面后仍可保留。
- 纯前端项目：无需服务器，只需直接打开 `index.html` 即可运行。

## 可能的扩展玩法

你可以基于当前项目继续扩展更多模式：

- 添加双人对战模式；
- 增加障碍物或墙体；
- 实现不同难度级别（速度、食物数量、地图大小）；
- 加入计时挑战模式；
- 允许玩家选择皮肤颜色或道具效果。
